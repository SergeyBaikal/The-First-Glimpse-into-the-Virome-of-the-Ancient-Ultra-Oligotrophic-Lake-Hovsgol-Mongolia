# The-First-Glimpse-into-the-Virome-of-the-Ancient-Ultra-Oligotrophic-Lake-Hovsgol-Mongolia
The repository contains supplementary materials and FASTA files obtained in the study.



1. The **AMG** folder contains 26 detected genes using DRAM-v. AMG_nucleotide.fasta contains the nucleotide sequences, and AMG_proteins.fasta contains the protein sequences.
2. The **NCLDV** folder contains 13 NCLDV bins identified in this study using the BEREN software.  
   PolB.fasta contains 6 detected PolB sequences from the bins.  
   Rhodopsin.fasta contains 2 rhodopsin proteins identified in the bins.
3. The **TerL** folder contains 2,115 sequences of the large terminase subunit protein from Lakes Hovsgol, Baikal, Biwa, and Tanganyika, used for network construction.
4. The **Full_genomes_phages** folder contains complete phage genomes, their TerL sequences, as well as the results of analysis performed with CheckV.
5. The **Viral_contigs** folder contains 10,895 viral contigs and the annotation results from tBLASTx against the RefSeq database (v.230).
6. The **Virophages** folder contains MCP amino acid sequences from Lake Hovsgol, identified both in virophage genomes and in contigs shorter than 10,000 nucleotides.
